## 功能特色

### 動態語音頻道

支援自由創建臨時語音頻道功能。用戶只需加入特定的語音頻道，機器人就會自動創建一個新的語音頻道。

- 自動創建與刪除臨時語音頻道
- 創建者擁有管理權限
- 所有成員可編輯頻道狀態

詳細設置請參閱：[動態語音頻道設置指南](./DYNAMIC_VOICE_SETUP.md)
